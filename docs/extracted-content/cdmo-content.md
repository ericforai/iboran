<span style="color:#ff0000"> __>案例客户按照项目管理的模式__ </span> 进行产品研发、项目计划、项目采购、项目生产、以及项目成本的核算，同时一个项目还可能跨地域在多工厂进行，进一步加大了管理协同的复杂程度。

<span style="color:#ff0000"> __>目前多个项目__ </span>  <span style="color:#ff0000"> __/__ </span>  <span style="color:#ff0000"> __管线并行生产，产品生产交叉进行，生产变更频繁。__ </span> 生产部门要在研发部门完成研发后，开始生产，生产准备周期较长。同时制造过程中不可预测的因素多，项目计划和任务的调整和变化、变更、例外情况发生是经常性的，对整个企业的生产计划排产，采购计划下达等都造成一系列管理困难。

<span style="color:#ff0000"> __>项目计划管理复杂。__ </span> 公司由于是项目制生产，案例客户往往会以每一个项目交货期为后墙倒排研发计划、采购计划、生产计划、检测计划，同时由于多个项目并行以及计划变更拖期等因素共同影响造成项目计划管理与追踪困难，项目交期延期是企业面临的典型的管理问题。

<span style="color:#ff0000"> __>相关资源的合理配套是生产管理的难点。__ </span> 由于各种变更因素存在，采购到料，协同工厂生产进度、生产设备目前状态，都会直接影响项目生产交付。同时还可能采用预投、库存计划、提前采购等手段保证交付，各个项目资源的配套及时性是企业在车间生产组织上面临的管理难点。

<span style="color:#ff0000"> __>生产制造有明确的时间和成本限制。__ </span> 在签订订单合同时，常常明确地规定了严格的交货时间和价格，如果企业延期交货对企业会造成多方面的影响，同时，成本的管理由以前核算到项目总体成本\(经常需要跨组织核算成本\)到目前按照利润中心的管理考核都给成本核算工作带来繁重的工作量和复杂性。

<span style="color:#e60012"> __案例客户关键需求 \- 项目管理难点__ </span>

<span style="color:#ff0000"> __>项目进度追踪管理：__ </span> 目前项目延误比较多，很多项目还是跨地域生产，集团项目管理部门没有统一的集团范围的项目管理平台无法通过统一的平台了解项目到底在哪一步延误了，是在哪个环节和哪个工厂延误了，目前这些信息分散公司管理不便。

<span style="color:#ff0000"> __>__ </span>  <span style="color:#ff0000"> __项目计划进度管理：__ </span> 需要在系统实现跨组织的统一平台的项目进度管控，需要实现项目状态的各种查询\(按照权限\)\,需要实现对研发计划的完成进度的统计管理\(工时管理\)。

<span style="color:#ff0000"> __>项目计划调整与排程：__ </span> 如果项目发生延误，在清楚掌握项目进度的情况下，延误需要要追回来，需要知道那家工厂延误，延误时间，然后集团重新调整生产计划，调整设备产能和物料是企业关心的问题。

<span style="color:#ff0000"> __>__ </span>  <span style="color:#ff0000"> __项目成本跨地域核算问题：__ </span> 项目成本核算费时费力，目前案例客户很多项目是跨地域协同生产，项目在不同地域不同工序进行生产，跨地域的生产数据\(领料、工时、制造费用\)难以获取，造成项目成本核算管理问题这种跨组织的项目成本核算问题需要解决。

<span style="color:#ff0000"> __>__ </span>  <span style="color:#ff0000"> __项目成本分析：__ </span> 项目有项目预算，项目完成后实现项目成本核算，需要进行项目成本分析，实现项目成本预实对比。

<span style="color:#e60012"> __案例客户关键需求__ </span>  <span style="color:#e60012"> __ \- __ </span>  <span style="color:#e60012"> __采购管理难点__ </span>

<span style="color:#ff0000"> __>集团采购管理问题：__ </span> 目前公司有集团采购的多种管理模式分别是：集中采购\-分散收货\-集中结算、集中采购\-分散收货\-分开结算、集中采购\-集中收货\-集中结算、目前系统无法支撑这种管理模式。

<span style="color:#ff0000"> __>供应商资质和供应商评级问题：__ </span> 企业有严格的供应商准入管理流程，供应商证照需要管理，不在供应商列表中的物料，不能购买。同时资质需要每年都需要进行评级，如何对供应商分级\(自己定义，按质量定义\)进行系统自动评价是目前难点。

<span style="color:#ff0000"> __>业务单据推送管理：__ </span> 目前U8系统更多是拉式生单，希望有更多的推式生单，加快生单效率。同时希望通过消息推送加强采购部门与需求部门的协同。

<span style="color:#ff0000"> __>__ </span>  <span style="color:#ff0000"> __采购计划变更频繁问题：__ </span> 项目计划变化频繁，导致采购计划变更频繁，需要系统支持针对计划变更频繁的管理计划模式，解决插单、改单等管理问题。建议关注专用料的变化利用LRP单独进行采购计划的排定。

<span style="color:#ff0000"> __采购预算管理问题：__ </span> 每月编制采购预算，每月预算是不准确的，预算控制到部门，控制总额。超预算走临时采购，公司临时采购现象频繁，区别预算内采购的临时采购流程是企业需要关注的。项目预算要求控制到料品，多个项目有公用料品时，会存在挪料的情况，需要系统支持项目物料控制等功能。

<span style="color:#ff0000"> __>  采购进度追踪管理问题：__ </span> 未来应用项目管理实现采购全流程的管理追踪。

<span style="color:#ff0000"> __>采购付款管理问题：__ </span> 需要从财务应付查询哪些订单已付，哪些未付的明细列表；在付款审批时，既要看总表审批又要看分项进行审批，需要支持批量审批重点是如何加强资金支出控制提高企业资金使用效率是企业关心的重点

<span style="color:#e60012"> __案例客户关键需求 \- 物料控制与管理难点__ </span>

<span style="color:#ff0000"> __>先入库再质检管理模式问题：__ </span> 公司物料都是先入库再质检，需要系统支持这种模式。同一批物料需要两个部门同时检验，两个部门同时判定合格后才可以视为质量合格，需要系统支持这种管理模式。

<span style="color:#ff0000"> __>物料预留__ </span> ：希望能按照项目预留库存，目前项目预留处理麻烦。未来通过系统可以实现按照项目采购入库后自动按照项目预留。需要系统支持项目关闭项目预留失效的功能。>物料超领控制管理：由于项目变化在生产一开始就可能超预算领料，如果材料不在BOM中，又要超领出来怎么管理。未来建议希望变更订单和BOM管理加强物料变更的管理规范性建立更合理的管理流程。

<span style="color:#ff0000"> __>各种库存预警机制建立：__ </span> 保质期预警、库龄预警、检测时间预警、滞库预警、安全库存预警，最低最高库存预警，需要系统建立灵活的预警方式。

<span style="color:#ff0000"> __>条码控制：__ </span> 希望未来实现物料出入库条码管理，以后扩展到产供销及单据条码；  \(需要使用无线条码功能，其中一个仓库需要考虑防爆无线产品\);

<span style="color:#ff0000"> __>进出口业务管理：__ </span> 公司99%的产品都是出口，目前报关是委托报关，未来需要系统支持装箱单、发票、报关单、等出口单据与业务流程。

<span style="color:#ff0000"> __>库存查询：__ </span> 需要系统支持按照项目号进行库存查询，需要支持跨组织库存查询。

<span style="color:#e60012"> __案例客户关键需求 \- 财务核算与管控难点__ </span>

<span style="color:#ff0000"> __预算管理问题：__ </span> 目前U8系统不支持预算管理功能，未来需要系统支持全面预算，包括费用预算、固定资产预算、采购预算、项目预算等预算管理与控制分析。

<span style="color:#ff0000"> __固定资产管理问题：__ </span> 希望系统实现大修、更新改造等、购置预算，同时支持固定资产购买报批、合同、付款等流程，同时实现合同的管理与追踪。同时集团能够统计全集团范围的固定资产情况；

<span style="color:#ff0000"> __项目预算管理问题：__ </span> 每年改造项目比较多\(改造可能是设备改造、设备维修等\)\,需要项目预算控制在每个项目改造时将钱花在哪些地方、花了多少，对预算有完整控制和分析。

<span style="color:#ff0000"> __费用预算管理问题：__ </span> 费用预算要对办公、差旅费等进行管控，没有预算或超过预算不准报销，对各个部门有费用考核；目前费用报销采取手工管理流程不透明与财务不集成，未来需要系统支持网上费用报销管理，公司存在跨组织费用报销管理情况。
