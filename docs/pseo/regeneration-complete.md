## 🔧 技术改进

### 1. Schema 生成差异化
- 根据关键词类型（ERP/MES/WMS/集成开发）自动生成不同的模块结构
- 每个类型有独特的内容模块，提供真实的信息增量

### 2. Page Renderer Fallback
- 当 raw_data 中缺少新字段时，自动从 mock_data.json 补充
- 如果 mock_data 也没有，使用类型特定的 fallback 逻辑
- 不因缺失字段而报错退出，确保页面能够生成

### 3. 类型特定的 Fallback
- `business_process` → 业务流程梳理通用说明
- `production_planning` → 生产计划流程通用说明
- `layout_design` → 设计流程通用说明
- `api_integration` → 集成对接流程通用说明
