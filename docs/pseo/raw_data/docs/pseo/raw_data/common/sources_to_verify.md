# sources_to_verify（对外主张核验清单）

本数据包的 company_profile 与 claims_company 默认按“待核验”处理：
- 如需对外发布，请用《泊冉公司介绍.pptx》或官网/合同等可引用材料核验后，将 verify_required=true 的条目改为 false。
- 建议维护一份“可公开版”（删掉客户名单/规模数据）与“内部版”（含全量证据）。

你需要补充/上传：
1) 泊冉公司介绍 PPT（用于自动抽取引用页码/原文）
2) 可公开的客户案例素材（行业、场景、截图、可引用描述）
3) 服务条款/交付范围模板（用于统一“我们做什么/不做什么”）
