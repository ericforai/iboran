# 去重报告

## 🔍 发现的问题

在数据库中发现 8 个重复标题的 Posts：

1. **WMS 实施服务** - 2 个
2. **MES 实施服务** - 2 个
3. **致远协同实施与定制服务** - 3 个
4. **ERP 二次开发服务** - 3 个
5. **NC 实施服务** - 2 个
6. **U8C 实施服务** - 2 个
7. **用友 YonBIP 实施与定制服务** - 2 个
8. **用友 YonSuite 实施服务** - 2 个

## 🔧 解决方案

### 去重策略

1. **保留最新版本**：按创建时间排序，保留最新的，删除旧的
2. **自动去重脚本**：`scripts/remove-duplicate-posts.ts`

### 使用方法

```bash
# 检查并删除重复的 Posts
npx tsx scripts/remove-duplicate-posts.ts
```

### 去重逻辑

- 按标题精确匹配分组
- 如果同一标题有多个 Posts，保留创建时间最新的
- 删除其他重复的 Posts
- 忽略 revalidate 警告（不影响删除成功）

## ✅ 去重结果

- 共发现 8 个重复标题
- 每个重复标题保留 1 个（最新的）
- 删除其他重复项

## 📋 预防措施

### 批量导入时避免重复

1. **导入前检查**：检查是否已存在相同标题的 Post
2. **使用唯一标识**：使用 slug 作为唯一标识
3. **更新而非创建**：如果已存在，更新而非创建新的

### 改进建议

在 `pseo-import-to-blog.ts` 中添加：
- 导入前检查是否已存在相同标题的 Post
- 如果存在，更新而非创建
- 或者跳过已存在的 Post
