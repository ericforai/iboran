# 转化系统设计规范

## CTA 设计原则

### 全站仅保留两类 CTA

| 类型 | 目的 | 示例 |
|------|------|------|
| 低门槛 | 留资获取内容 | "获取方案白皮书"、"下载架构图" |
| 高门槛 | 高意向筛选 | "预约专家演示" |

### 路径约束
```
任意页面 → 表单提交 ≤ 2 次点击
```

---

## CTA 放置规则

### 导航栏
- 右侧固定红色按钮："预约演示"
- 使用 `DemoRequestModal` 组件

```tsx
<button className="bg-[#E60012] text-white px-6 py-2.5 rounded-md">
  预约专家演示
</button>
```

### Hero 区块
- 主 CTA：高对比红色按钮
- 次 CTA：蓝色边框按钮

### 方案页底部
- 固定底部悬浮 CTA 条
- 或大尺寸 CTA 区块

---

## 表单设计

### 必填字段
1. 姓名
2. 手机号（11位验证）
3. 公司名称

### 可选字段
- 行业选择
- 需求描述
- 预算范围

### 表单组件
使用 Payload Form Builder 插件：
```tsx
import { DemoRequestModal } from '@/components/DemoRequestModal'
```

## AI 客服分级逻辑

| 场景 | 用户行为 | 系统响应 |
|------|----------|----------|
| A | 浏览 < 30 秒 | 静默，仅显示悬浮入口 |
| B | 浏览方案页并滚动到底部 | AI 弹出："需要了解报价或实施周期吗？" |
| C | 提问涉及"价格"、"试用" | 停止 AI 回复，推送顾问企微或表单 |

---

## 浮动元素

### 悬浮咨询入口
- 位置：右下角固定
- 状态：默认收起
- 触发：用户主动点击或满足场景 B 条件

### 电话热线
- 导航栏展示：`400-9955-161`
- 带电话图标悬浮效果
