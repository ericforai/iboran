# 泊冉官网项目概览

## 项目定位

泊冉 (iboran.com) 是用友生态中的专业实施服务商，**不是**用友官网的复制品。

### 核心差异点
- **用友**：提供标准化产品
- **泊冉**：提供场景化落地与本地服务（二开/实施/咨询）

### 用户认知目标
访客能快速识别泊冉的独立服务商身份，并认为通过泊冉实施比直接联系原厂更专业、响应更快。

---

## 技术栈

| 层级 | 选型 | 理由 |
|------|------|------|
| 前端框架 | Next.js 15 (App Router) | SSR 支持，百度爬虫可直接抓取 |
| CMS | Payload CMS 3.x (自托管) | 内容与代码分离，API 批量导入 |
| 样式 | Tailwind CSS 3.x | 原子化 CSS，设计系统一致性 |
| 动画 | Framer Motion | 微交互与页面过渡 |
| 图标 | Lucide React | 轻量一致的图标库 |

---

## 目录结构

```
src/
├── app/
│   ├── (frontend)/    # 前端页面路由
│   └── (payload)/     # Payload 管理后台
├── blocks/            # 页面区块组件
├── collections/       # Payload 数据集合
├── components/        # 公共 React 组件
├── heros/             # 首屏 Hero 组件
├── Header/            # 导航头部
├── Footer/            # 页脚
└── utilities/         # 工具函数
```

## 可量化指标

1. **SEO**: 核心业务词进入百度首页，新页面收录率 > 90%
2. **内容扩展**: 支持 CMS 批量生产 200+ 方案页
3. **线索分级**: 高意向转人工，低意向进入自动培育
