# 泊冉软件 (iboran.com)

> 用友 YonBIP / YonSuite 官方实施与定制服务商

## 项目简介

泊冉软件（上海）有限公司是用友网络合作伙伴，专注于为企业提供 YonBIP、YonSuite 等产品的实施、定制和咨询服务。

本网站是基于 Next.js 15 和 Payload CMS 3.x 构建的营销展示网站，展示产品解决方案、成功案例和技术服务。

## 技术栈

- **前端**: Next.js 15 (App Router), React 19, TypeScript
- **CMS**: Payload CMS 3.x
- **数据库**: MongoDB 7
- **样式**: Tailwind CSS 3.x
- **AI**: DeepSeek API

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 访问
# http://localhost:3000 - 前端网站
# http://localhost:3000/admin - Payload 管理后台
```

## 主要功能

- **AI 智能顾问**: 基于 DeepSeek API 的智能问答，支持知识库检索和人工转接
- **产品展示**: YonSuite、YonBuilder、MDM、BPM 等产品介绍
- **解决方案**: 行业解决方案和业务场景
- **成功案例**: 客户案例和实施经验
- **在线咨询**: 表单和电话咨询入口
- **数据分析**: Google Analytics 和百度统计集成
